# KidGuard — YouTube Parental Control Extension

A Chrome extension that blocks inappropriate keyword searches and videos on YouTube. Password-protected so only parents can change the settings.

## Features

- 🔐 **Password protected** — children cannot access or change settings
- 🚫 **Block search keywords** — prevents searching for blocked terms
- 📹 **Block video pages** — hides videos whose titles contain blocked words
- 🏠 **Hide results** — blocked videos are hidden from search results and homepage
- ⚡ **Real-time** — blocks as you type before the search is submitted
- 🏷 **Quick presets** — one-click add common inappropriate keywords
- 🟢 **Toggle on/off** — temporarily pause filtering with a click

## Installation

1. Extract this ZIP file
2. Open Chrome → go to `chrome://extensions/`
3. Enable **Developer Mode** (top-right toggle)
4. Click **"Load unpacked"** → select the `kidguard` folder
5. The KidGuard shield icon appears in your toolbar

## First Use

1. Click the KidGuard icon in your toolbar
2. **Set a parent password** — this protects the settings
3. Add keywords you want to block (e.g. inappropriate words)
4. Use the **Quick Add presets** for common blocks
5. Protection is active immediately on YouTube

## How It Blocks

- **Search bar**: If a blocked keyword is typed and Enter is pressed, the search is cancelled and a warning shown
- **Video pages**: If you navigate to a video whose title contains a blocked word, a full-screen block page appears
- **Results & homepage**: Videos with blocked titles are hidden from all lists

## Updating Keywords

Click the extension icon → enter your parent password → add or remove keywords. Changes apply instantly.
